# CorrelatedCounts
Modeling correlated count data based off of the model in Rodrigues-Motta et al 2013.

Mariana Rodrigues-Motta, Hildete P. Pinheiro, Eduardo G. Martins, Márcio S. Araújo & Sérgio F. dos Reis (2013) Multivariate models for correlated count data, Journal of Applied Statistics, 40:7, 1586-1596, DOI: 10.1080/02664763.2013.789098
